Homework 9, Part 1: Bubbles charts and force diagrams

billionaires_complete.csv is a nice long list of billionaires. Let's make some charts with it!

You will probably find these video tutorials useful in addition to the completed classwork up on CourseWorks.

Feel free to resize the svg elements! They're kind of small and square.

Ask questions in #storytelling, as always.

Chart 1: A bubble chart

Make a bubble chart of every billionaire. Size each circle by 'net' - their net worth.

Create and use images for at least 10 billionaires. You can just steal them from Google.

Chart 2: Interacting

Make a bubble chart of every billionaire. Size each circle by 'net' - their net worth. Color code by industry - you'll also be including the image'd circles from Chart 1, so they'll need outlines in the given color.

Add a series of buttons to allow you to split the bubbles into groupings - by gender and by industry would be nice - and a button to return them all to the middle. You can space out the gender one by hand, but use a scale for industry. Each industry should include a label.

Extra credit: add tooltips to the circles

Chart 3: Mixing it up

Make a scatterplot of every billionaire. Age on the x axis, net worth on the y axis. Color code by industry. Include a button that, when clicked, turns the graphic into a force diagram clustered around countries. When the button is clicked, the axes should disappear and labels for each country should appear instead. Also include a button to revert back to the scatterplot.

Once you've completed all of these, go into the CSS up top and stop the SVG elements from having a grey background.
